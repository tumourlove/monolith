F16 rewrote TC2.9/TC2.11 (and §Setup ability fixtures) to use the existing `Ability.Combat.Melee.Light/Heavy` registry tags; Punch/Kick deliberately not in tree per survival-horror curation.
4. **`add_bt_smart_object_task` blocked on `GameplayBehaviorSmartObjects` plugin enable + editor restart**. Source review confirms F1 hardening present at `MonolithAIBehaviorTreeActions.cpp:3071`; cannot exercise end-to-end without plugin.
